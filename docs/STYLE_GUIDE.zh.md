# 风格指南

每个纪元有独特的文风。所有散文（Markdown 文件）应与其所属纪元的风格一致。

## 各纪元文风

### epoch:01 — 混沌（太初混沌）
- 风格：神话、庄严、古朴
- 语言：宏大、宣言式的句子。极少对话。创世神话的语域。
- 意象：虚空、无形、宇宙尺度、诞生与牺牲
- 避免：现代词汇、随意语气、技术性语言

### epoch:02 — 太初（天地初开）
- 风格：元素感、原始、磅礴
- 语言：感官描写——火、水、风、土。原始力量。
- 意象：黎明、光破黑暗、元素碰撞
- 避免：政治语言、社会结构描写

### epoch:03 — 神荒（野神时代）
- 风格：狂野、灵性、不羁
- 语言：万物有灵、敬畏自然。神灵有个性且反复无常。
- 意象：山林、河流皆为活物。神灵行为不可预测。
- 避免：官僚语言、系统化思维

### epoch:04 — 万国（文明崛起）
- 风格：史诗、政治、多元
- 语言：历史编年体风格。多视角叙事。外交与军事词汇。
- 意象：城池、军队、商路、文化交流
- 避免：神话语域（神灵已远去）、现代俚语

### epoch:05 — 礼法（秩序时代）
- 风格：正式、官僚、规范
- 语言：法律与哲学语域。精确、结构化的散文。
- 意象：朝堂、典籍、礼仪、等级制度
- 避免：混乱、狂野、非正式表达

### epoch:06 — 金潮（黄金浪潮）
- 风格：商业化、讽刺、超现代
- 语言：营销话术、反讽、自我意识。资本主义隐喻。
- 意象：市场、契约、广告、万物商品化
- 避免：敬畏、庄严（一切皆可买卖）

### epoch:07 — 元寂（元叙事寂灭）
- 风格：存在主义、元叙事意识、哲学性
- 语言：学术性、内省式。角色讨论叙事结构本身。
- 意象：模式、递归、故事中的故事、虚构的边界
- 避免：简单的英雄主义、直白的叙事（一切都被质疑）
