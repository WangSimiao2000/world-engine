# 纪元指南

## epoch:01 — 混沌 Hùn Dùn（太初混沌）

**主题**：天地未分，万物未生。

世界始于未分化的混沌。第一个创世之举——盘古开天辟地——确立了天与地的根本二元。此纪元以盘古之死及其化身为物质世界而终结。

**关键约束**：
- 没有现代意义上的文明（仅有如元灵族这样的原初意识体）
- 没有技术，没有社会结构
- 角色是宇宙尺度的存在，而非人类
- 事件是唯一的、不可逆的、定义世界的

## epoch:02 — 太初 Tài Chū（天地初开）

**主题**：元素初生，天地成形。

原始元素开始成形。火、水、风、土作为独立力量出现。世界获得了物质形态，但仍然狂野而无序。

**关键约束**：
- 元素力量占主导
- 尚无有组织的社会
- 角色为元素精灵或原始存在

## epoch:03 — 神荒 Shén Huāng（野神时代）

**主题**：万灵争鸣，神灵横行。

精灵与野神自由游荡。世界是万物有灵的——每座山、每条河、每片森林都有灵。神灵强大而反复无常，但并未组织化。

**关键约束**：
- 神灵直接干预但行为混乱
- 没有集中的神权
- 凡人开始出现但微不足道

## epoch:04 — 万国 Wàn Guó（文明崛起）

**主题**：人族崛起，万国林立。

凡人组建文明。神灵退隐。政治结构、贸易、战争和文化交流定义了这个时代。

**关键约束**：
- 神灵已远——极少直接干预
- 多个文明共存与竞争
- 技术水平为前工业时代（青铜/铁器时代）

## epoch:05 — 礼法 Lǐ Fǎ（秩序时代）

**主题**：秩序至上，礼法森严。

一个主导文明将法律、礼仪和官僚体系编纂成典。知识被系统化。世界变得有序但僵化。

**关键约束**：
- 中央集权
- 正式的哲学与法律体系
- 创造力受到规则约束

## epoch:06 — 金潮 Jīn Cháo（黄金浪潮）

**主题**：万物皆可买卖。

超级商业化。一切——包括灵力、神器，甚至叙事套路——都被商品化。这是资本主义隐喻的纪元。

**关键约束**：
- 经济体系主导生活的方方面面
- 讽刺与反讽是合适的表达方式
- 早期纪元的神圣之物如今都是商品

## epoch:07 — 元寂 Yuán Jì（元叙事寂灭）

**主题**：世界发现自己是一个故事。

最先进的文明发现世界运行的是叙事法则，而非物理法则。这引发了一场存在主义危机，最终导致世界的崩塌与重置。

**关键约束**：
- 元叙事意识是核心——角色讨论叙事结构本身
- 元层级（`world/meta/`）主要与此纪元关联
- 元注释中的 `observer` 应为此纪元的文明
- 此纪元以崩塌终结——没有「大团圆结局」
